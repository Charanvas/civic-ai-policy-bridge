# 🏛️ Civic AI Policy Bridge

An AI-powered civic intelligence platform designed to bridge the communication gap between governments, policies, and citizens using accessible AI-driven policy interpretation and engagement systems.

---

# 🌟 Overview

Civic AI Policy Bridge is a public-interest AI platform that simplifies complex government policies, civic schemes, legal information, and administrative procedures into understandable, actionable insights for citizens.

The platform combines:
- AI-powered policy interpretation
- Civic information assistance
- Public service accessibility
- Multilingual support
- Intelligent citizen engagement
- Government communication simplification

The mission is to make governance understandable, transparent, and accessible for everyone.

# 🚀 Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Charanvas/civic-ai-policy-bridge.git
cd civic-ai-policy-bridge
```

---

## 2️⃣ Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / Mac

```bash
source venv/bin/activate
```

---

## 3️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 4️⃣ Configure Environment Variables

Create a `.env` file:

```env
OPENAI_API_KEY=your_api_key
OCR_ENGINE=tesseract
DATABASE_URL=your_database_url
```

---

## 5️⃣ Run Application

```bash
python app.py
```
